# House Price Prediction using Linear Regression

## Overview

This project predicts house prices using a Machine Learning Linear Regression model.

## Features

* Square Footage
* Number of Bedrooms
* Number of Bathrooms

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Steps

1. Load the dataset
2. Split data into training and testing sets
3. Train a Linear Regression model
4. Predict house prices
5. Evaluate model performance using R2 Score and Mean Absolute Error

## Output

* R2 Score
* Mean Absolute Error
* Predicted House Price
